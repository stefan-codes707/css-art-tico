Introducción

Objetivos

Con este taller de aprender CSS artístico, se tiene como objetivo principal, motivar el aprendizaje creativo del desarrollo web, especialmente en el área del CSS, para poder expandir conocimientos através de dibujos y logos.

Al igual que promover un nuevo planteamiento o estrategia de aprendizaje en el entorno de desarrollo web, donde se pueda aprender CSS de una forma nueva y creativo, insentivando a los futuros programadores.

Metas

Como cualquier curso o taller en el área de programación, se tiene como meta principal la adquisición de conocimientos nuevos, al igual que con este taller queremos dar a conocer que hay nuevas maneras y mucho más creativas de aprender CSS sin que sea tan complejo.

Queremos que al culminar este taller tengas conocimientos sobre:

  • Propiedades básicas de CSS
  • Animaciones
  • Responsive Design
  • JavaScript básico
  • Preprocesador Sass
  • Libreria Bootstrap

Pixel Art

Con el Pixel Art aprenderemos la propiedad box-shadow, una de las propiedades que a menudo se usa cuando hacemos cartas de presentación, botones, entre otros elementos. Al hacer un Pixel Art aprenderemos a fondo esta propiedad.

5

Responsive Design

La parte más esencial cuando estamos haciendo una página web, es adaptar la página web a distintas definiciones, como celulares, tablets. O, que creías? ¿Que se hacía solo? Pues no, hay que adarptarlo y para eso existen los media-queries, una herramienta que nos ayudará a modeldear la página web.

6

JavaScript

JavaScript le agregará dinamismo a nuestra página web para que haga distintas funcionalidades y podamos completar nuestro sitio web.

En el taller aprendemos comandos básicos del lenguaje de programación JavaScript, tales como funciones, eventos, variables y ciclos. Si no tienes conocimietos previos, no te preocupes, se explicará detalladamente cada proceso.

Veremos para que funciona, para que sirve y haremos alguna que otra práctica.
